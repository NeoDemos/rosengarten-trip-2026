# Rosengarten Trip 2026

Mobile-friendly GitHub Pages version of the Rosengarten / Catinaccio hut-to-hut plan for June 2026.

## Sync

The published page is `index.html`. After editing, commit and push `main`; GitHub Pages serves the latest version from that branch.

If you keep editing the original local file at `../Rosengarten_trip_juni_2026.html`, run:

```bash
./publish.sh
```

That copies the latest local trip file into `index.html`, commits it, and pushes it to GitHub Pages.
